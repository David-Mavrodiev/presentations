Linear Regression Explained

Linear regression is a statistical method used in machine learning for predicting a continuous dependent variable based on one or more independent variables. It assumes a linear relationship between the input(s) (independent variables) and the single output (dependent variable).

How to find the optimal weights/coefficients?

Solving for Weights with Respect to Error

1. Equation with the error

Y = WX + ε

Where Y is the vector of observed values, X is the matrix of input features, W is the vector of weights, and ε represents the error terms.

2. Objective - Minimize the Error

The goal is to find the weights W that minimize the error between the predicted values (WX) and the actual values (Y). This is done by minimizing the error (MSE).

3. Solve the Normal Equation

The normal equation for solving weights using MSE is derived as setting the derivative of the MSE with respect to the weights to zero.

W = (XTX)−1XTY

Gradient descent

Gradient descent is a method to find the optimal coefficients/weights in linear regression that minimize error (like MSE).

1. Starting with random coefficient values.

y = θ0 + θ1x1 + θ2x2 + ... + θnxn

2. Calculating the gradient of the error function with respect to each coefficient.

∂/∂θj MSE(θ) = (2/m) ∑ ((θT x(i) - y(i)) xj(i))

This represents the partial derivative of the MSE with respect to the coefficient θj. x(i) is the feature vector of the i-th training example, y(i) is its target value, and xj(i) is the value of feature j for this example.

3. Updating the coefficients by subtracting a portion of the gradient, scaled by a learning rate.

θj := θj - α (∂/∂θj MSE(θ))

Here, θj is updated by subtracting the product of the learning rate α and the gradient of the MSE with respect to θj.

4. Repeating the process until the algorithm converges.

Training

Types of Gradient Descent

Batch Gradient Descent

Uses the entire training dataset.

Computes the gradient of the cost function with respect to the parameters for the entire training dataset.

  • Very stable convergence.
  • Can handle noisy data well.
  • Computationally expensive for large datasets.

Stochastic Gradient Descent (SGD)

Uses a single data point (or instance) at each iteration.

The gradient is calculated and parameters are updated for each data point separately.

  • Much faster than batch gradient descent.
  • Can handle very large datasets efficiently.
  • The path to convergence is noisier compared to batch gradient descent.

Mini-batch Gradient Descent

Uses a subset (mini-batch) of the training dataset.

The gradient is computed and the parameters are updated for each mini-batch of data.

  • Balances the efficiency of SGD with the stability of BGD.
  • Less erratic update path compared to SGD.
  • Batch size must be carefully chosen.

Linear Regression Variations

Ridge Regression

Adds a regularization term equal to the square of the magnitude of the coefficients (L2 regularization). This helps to prevent overfitting by penalizing large coefficients.

Cost function = Σ(yi - yi)² + λ Σ wj²

Where yi is the actual value, yi is the predicted value, wj are the coefficients, and λ is the regularization parameter.

Lasso Regression

Similar to Ridge Regression, but the regularization term is the absolute value of the magnitude of the coefficients (L1 regularization). This can lead to some coefficients being exactly zero, which is useful for feature selection.

Cost function = Σ(yi - yi)² + λ Σ |wj|

The notation is similar to Ridge Regression, but the regularization term is the sum of the absolute values of the coefficients.

Polynomial Regression

Polynomial regression is a form of regression analysis in which the relationship between the independent variable and dependent variable is modeled as an nth degree polynomial, enabling it to fit a wide range of curvature and complexity in the data.

y = θ0 + θ1x1 + θ2x2 + θ3x12 + θ4x22 + θ5x1x2 + ...
