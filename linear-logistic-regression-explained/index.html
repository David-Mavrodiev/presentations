Linear Regression

Linear regression is a statistical method used in machine learning for predicting a continuous dependent variable based on one or more independent variables. It assumes a linear relationship between the input(s) (independent variables) and the single output (dependent variable).

y = θ0 + θ1x1 + θ2x2 + ... + θnxn

y = θ0 + θ1x

y = θ0 + θ1x1 + θ2x2

How to find the optimal weights/coefficients that best fits the given set of data points?

Measure error

Mean Squared Error (MSE)

MSE is the average of the squares of the errors—that is, the average squared difference between the estimated values and the actual value.

Gradient descent

Gradient descent is a method to find the optimal coefficients in linear regression that minimize error (like Mean Squared Error).

1. Starting with random coefficient values.

y = θ0 + θ1x1 + θ2x2 + ... + θnxn

2. Calculating the gradient of the error function with respect to each coefficient.

∂/∂θj MSE(θ) = (2/m) ∑ ((θT x(i) - y(i)) xj(i))

This represents the partial derivative of the MSE with respect to the coefficient θj. x(i) is the feature vector of the i-th training example, y(i) is its target value, and xj(i) is the value of feature j for this example.

3. Updating the coefficients by subtracting a portion of the gradient, scaled by a learning rate.

θj := θj - α (∂/∂θj MSE(θ))

Here, θj is updated by subtracting the product of the learning rate α and the gradient of the MSE with respect to θj.

4. Repeating the process until the algorithm converges.
